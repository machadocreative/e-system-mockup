# e-system — UI/UX Mockup

Maquete estática (modernizada) da interface do **e-system**, para revisão visual em equipe.

- **Arquivo único:** `index.html` (CSS, JS e ícones embutidos; sem build, sem dependências além da fonte do Google Fonts via CDN).
- **Como ver:** abra `index.html` no navegador (Chrome/Safari) ou acesse a versão publicada (GitHub Pages).
- **Telas:** use o seletor no topo — Login · Dashboard · Articles · Article Form · Newsletter · Photos (o Album abre ao clicar numa foto).

> ⚠️ Este repositório contém **apenas a maquete**. Nenhum código, banco de dados ou
> credencial do sistema real está aqui (e o `.gitignore` bloqueia esses arquivos por segurança).
